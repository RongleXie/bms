# BMS 博客管理系统 - 发布说明

## v1.0.0 (2026-03-25)

### 🎉 首次正式发布

BMS（Blog Management System）是一个基于 Snowy v3.6.1 框架开发的轻量级博客管理系统，提供完整的博客内容管理解决方案。

## 🚀 快速启动

### 1. 数据库初始化

```bash
# 创建数据库
mysql -u root -p -e "CREATE DATABASE bms DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_general_ci;"

# 执行初始化脚本
mysql -u root -p bms < .planning/sql/bms_init.sql
mysql -u root -p bms < .planning/sql/bms_menu_permission.sql
```

### 2. 后端启动

```bash
# 编译打包
mvn clean install -DskipTests

# 启动后端
java -jar bms-web-app/target/bms-web-app-3.0.0.jar
```

### 3. 前端启动

```bash
cd bms-admin-web
npm install
npm run dev
```

### 4. 访问系统

- 前端地址: http://localhost:81
- 后端地址: http://localhost:82
- API文档: http://localhost:82/doc.html
- 默认账号: superAdmin / 123456
