## 📁 项目结构

```
bms/
├── bms-common/              # 公共模块
├── bms-plugin-api/          # 插件API
├── bms-plugin/              # 插件实现
│   └── bms-plugin-biz/      # 业务插件
├── bms-web-app/             # 主启动模块
├── bms-admin-web/           # 前端Vue3项目
├── .planning/               # 项目规划文档
│   ├── docs/                # 设计文档
│   ├── sql/                 # SQL脚本
│   │   ├── bms_index_optimization.sql    # 索引优化脚本 (新增)
│   │   └── bms_counter_sync_timer.sql    # 计数器同步定时任务 (新增)
│   ├── prototype/           # 原型页面
│   └── ROADMAP_V2.md        # V2.0.0规划
├── docs/                    # 审查报告
├── RELEASE_NOTES.md         # 发布说明
```
