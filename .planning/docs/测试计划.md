#### 单元测试模板

```java
/**
 * BmsArticleServiceImpl单元测试
 */
@SpringBootTest
class BmsArticleServiceImplTest {

    @Resource
    private BmsArticleService bmsArticleService;

    @Resource
    private BmsArticleMapper bmsArticleMapper;

    @Test
    @DisplayName("新增文章-成功")
    void testAdd_Success() {
        // Given
        BmsArticleAddParam param = new BmsArticleAddParam();
        param.setTitle("测试文章标题");
        param.setContent("这是测试文章内容，长度满足要求。");
        param.setCategoryId("CATEGORY_ID_001");
        param.setStatus("DRAFT");
        
        // When
        bmsArticleService.add(param);
        
        // Then
        BmsArticle entity = bmsArticleMapper.selectOne(
            new LambdaQueryWrapper<BmsArticle>()
                .eq(BmsArticle::getTitle, "测试文章标题")
        );
        assertNotNull(entity);
        assertEquals("DRAFT", entity.getStatus());
    }

    @Test
    @DisplayName("新增文章-标题为空")
    void testAdd_TitleEmpty() {
        // Given
        BmsArticleAddParam param = new BmsArticleAddParam();
        param.setTitle("");
        param.setContent("测试内容");
        
        // When & Then
        assertThrows(ServiceException.class, () -> {
            bmsArticleService.add(param);
        });
    }

    @Test
    @DisplayName("新增文章-标题超长")
    void testAdd_TitleTooLong() {
        // Given
        BmsArticleAddParam param = new BmsArticleAddParam();
        param.setTitle("A".repeat(201)); // 超过200字符限制
        param.setContent("测试内容");
        
        // When & Then
        assertThrows(ServiceException.class, () -> {
            bmsArticleService.add(param);
        });
    }
}
```

## 8. 缺陷管理

### 8.1 缺陷等级定义

| 等级 | 名称 | 定义 | 处理时限 | 示例 |
|------|------|------|----------|------|
| P0 | 致命 | 系统崩溃、数据丢失、核心功能无法使用 | 立即修复 | 文章发布失败、数据库连接断开 |
| P1 | 严重 | 主要功能异常、安全漏洞 | 24小时内 | XSS攻击成功、分类删除失败 |
| P2 | 一般 | 次要功能异常、影响体验 | 72小时内 | 排序错误、提示信息不准确 |
| P3 | 轻微 | UI问题、文字错误 | 版本迭代 | 按钮样式偏差、错别字 |
