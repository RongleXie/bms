# 博客管理系统 (BMS) 数据库设计文档

## 1. 概述

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| 规范性 | 遵循Snowy框架表设计规范，统一公共字段 |
| 扩展性 | 预留扩展字段(ext_json)，支持业务变化 |
| 性能优化 | 合理设计索引，支持高频查询 |
| 可维护性 | 统一命名规范，完善注释 |

### 1.2 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 表名 | BMS_{实体名} 大写下划线 | BMS_ARTICLE |
| 字段名 | 大写下划线 | CREATE_TIME |
| 索引 | IDX_{表名}_{字段} | IDX_ARTICLE_STATUS |
| 唯一索引 | UK_{表名}_{字段} | UK_CATEGORY_CODE |

## 2. 表清单

| 序号 | 表名 | 说明 | 预估数据量 |
|------|------|------|------------|
| 1 | BMS_CATEGORY | 文章分类表 | 百级 |
| 2 | BMS_TAG | 文章标签表 | 百级 |
| 3 | BMS_ARTICLE | 文章表 | 万级 |
| 4 | BMS_ARTICLE_TAG | 文章标签关联表 | 十万级 |
| 5 | BMS_COMMENT | 评论表 | 十万级 |
| 6 | BMS_MEDIA | 媒体资源表 | 万级 |

## 3. 实体关系

```
BMS_CATEGORY (1) ----< (N) BMS_ARTICLE
BMS_TAG (N) >----< (N) BMS_ARTICLE (通过 BMS_ARTICLE_TAG)
BMS_ARTICLE (1) ----< (N) BMS_COMMENT
SYS_USER (1) ----< (N) BMS_ARTICLE
SYS_USER (1) ----< (N) BMS_MEDIA
```

## 4. 核心表设计

### 4.1 文章表 (BMS_ARTICLE)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| ID | VARCHAR(32) | 主键ID |
| TITLE | VARCHAR(200) | 文章标题 |
| SUMMARY | VARCHAR(500) | 文章摘要 |
| CONTENT | LONGTEXT | 文章内容 |
| COVER_IMAGE | VARCHAR(255) | 封面图片URL |
| CATEGORY_ID | VARCHAR(32) | 分类ID |
| AUTHOR_ID | VARCHAR(32) | 作者ID |
| STATUS | VARCHAR(20) | 状态(DRAFT/PUBLISHED) |
| VIEW_COUNT | INT | 阅读量 |
| LIKE_COUNT | INT | 点赞数 |
| COMMENT_COUNT | INT | 评论数 |
| IS_TOP | TINYINT | 是否置顶 |
| IS_RECOMMEND | TINYINT | 是否推荐 |
| PUBLISH_TIME | DATETIME | 发布时间 |

### 4.2 评论表 (BMS_COMMENT)

| 字段名 | 类型 | 说明 |
|--------|------|------|
| ID | VARCHAR(32) | 主键ID |
| ARTICLE_ID | VARCHAR(32) | 文章ID |
| PARENT_ID | VARCHAR(32) | 父评论ID |
| NICKNAME | VARCHAR(50) | 评论者昵称 |
| EMAIL | VARCHAR(100) | 评论者邮箱 |
| CONTENT | VARCHAR(1000) | 评论内容 |
| STATUS | VARCHAR(20) | 状态(PENDING/APPROVED/REJECTED) |

## 5. 索引策略

### 5.1 文章表索引

| 索引名 | 类型 | 字段 | 说明 |
|--------|------|------|------|
| PRIMARY | 主键 | ID | 主键索引 |
| IDX_ARTICLE_CATEGORY | 普通 | CATEGORY_ID | 分类查询 |
| IDX_ARTICLE_STATUS | 普通 | STATUS | 状态筛选 |
| IDX_ARTICLE_PUBLISH_TIME | 普通 | PUBLISH_TIME | 时间排序 |
| IDX_ARTICLE_TOP | 组合 | IS_TOP, PUBLISH_TIME | 置顶文章查询 |

### 5.2 评论表索引

| 索引名 | 类型 | 字段 | 说明 |
|--------|------|------|------|
| PRIMARY | 主键 | ID | 主键索引 |
| IDX_COMMENT_ARTICLE | 普通 | ARTICLE_ID | 文章评论查询 |
| IDX_COMMENT_STATUS | 普通 | STATUS | 状态筛选 |
