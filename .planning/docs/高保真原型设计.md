# 博客管理系统 (BMS) 高保真原型设计

## 原型概述

本项目采用零依赖的HTML/CSS/JS技术栈，创建高保真可交互原型，用于验证设计方案和演示交互效果。

## 设计风格

### 后台管理界面
- 基于Ant Design设计规范
- 简洁现代的商务风格
- 主色调：#1890ff（蓝色）
- 布局：左侧菜单 + 顶部导航 + 内容区域

### 前台展示界面
- 简洁现代的博客风格
- 卡片式布局
- 响应式设计
- 主色调：#1890ff（蓝色）+ 渐变色彩

## 交互说明

### 后台管理

1. **文章列表页**
   - 支持按标题、分类、状态搜索
   - 表格支持排序和批量操作
   - 操作按钮：编辑、查看、删除

2. **文章编辑页**
   - 富文本编辑器（支持工具栏）
   - 标签输入（支持回车添加）
   - 封面图片上传
   - SEO设置

### 前台展示

1. **博客首页**
   - 顶部导航栏（分类导航）
   - Banner区域（网站介绍）
   - 文章卡片列表
   - 侧边栏（分类、标签、热门文章）

2. **文章详情页**
   - 文章标题、元信息
   - 富文本内容展示
   - 目录导航
   - 评论系统
   - 相关文章推荐

## 技术实现

- **零依赖**：纯HTML/CSS/JS，无需任何外部库
- **响应式**：适配桌面端和移动端
- **可交互**：包含基本的hover效果和表单交互
- **模拟数据**：使用静态模拟数据填充
